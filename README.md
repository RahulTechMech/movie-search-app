# 🎬 Movie Search App

## 📌 Overview
This is a responsive single-page application built using React.js that allows users to browse and search movies using an external API.

## 🚀 Features
- 🔍 Search movies in real-time
- 🎬 Display movie cards with poster, title, rating, and description
- ⏳ Loading state while fetching data
- ❌ Error handling for API failures
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🧩 Component-based architecture

## 🛠 Tech Stack
- React.js
- Tailwind CSS
- Axios
- TMDB API

## 📂 Folder Structure
src/
 ├── components/
 ├── pages/
 ├── services/

## ⚙️ Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Add your API key in `api.js`
4. Run `npm run dev`
